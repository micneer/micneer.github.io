**Vim常用的移动光标的命令**

|  命令   |       作用      |
|:------:|:----------------|
| gg  | 命令将光标移动到文档开头 |
| G   | 命令将光标移动到文档末尾 |
| nG  | 移动到第n行上 |
| $   | 移动光标到行尾 |
| n$  | 移动到第n行的行尾 |
| 0（Num）  | 移动光标到行首 |
| ctrl + f | 向前（向下）翻页 |
| ctrl + b | 向后（向上）翻页 |
| ctrl + r | 恢复 |
| u | 撤销 |
| nyy | 复制多少行 |
| p   | 粘贴 |
