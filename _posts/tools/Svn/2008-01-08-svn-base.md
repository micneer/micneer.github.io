**SVN常用命令**

```xml

svn co svn://svn.com/repos repos
svn up 
svn add * --force
svn ci 

```

**添加externals**

```xml

svn propedit svn:external . --editor-cmd vim

```

在vim中编辑

*common*是需要引用的目录

```xml

^/repos/sideproject/xxxx/common common

```

**externals其它命令**

```xml

svn proplist
svn propget
svn propset

```
